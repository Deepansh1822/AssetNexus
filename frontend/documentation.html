Overview

AssetNexus is a full-stack enterprise asset management system built with Spring Boot (backend) and Vanilla JS + HTML5 (frontend). It enables organizations to track, assign, maintain, and report on physical assets across their workforce.

Java 17 Spring Boot 3 MySQL Spring Security HTML5/JS

User Roles

Administrator

  • Full dashboard access
  • Add, edit, delete assets
  • Assign & return assets
  • Manage employees
  • Manage categories
  • View all maintenance requests
  • Access full reports

Employee

  • View assigned assets only
  • Submit maintenance requests
  • View own request history
  • View personal reports
  • Update own profile
  • Cannot manage other employees
  • Cannot add or delete assets

Asset Management

Assets are the core entity of the system. Each asset has a unique auto-generated Asset Tag and can be tracked through its full lifecycle.

Categories

Categories group assets by type (e.g., Laptops, Furniture, Vehicles). Each category can be set as Active or Inactive. Inactive categories are hidden from asset creation forms.

Only Administrators can create, edit, or delete categories.

Employees

Employees are system users who can be assigned assets. Each employee has a role (ADMIN or EMPLOYEE) that determines their access level.

Deleting an employee will unassign all their assets. The assets will return to Available status.

Workforce Mobility

The Labour Management module is a high-precision registry for field personnel. It tracks skilled and unskilled workforce mobility across various construction projects.

Field Supplies

Field Supplies (Stocks) monitors mission-critical raw materials and inventory dispatched to construction sites. It includes high-precision alert triggers for low-stock thresholds.

Stock Tracking Parameters

FieldTypeDescription
materialNameStringName of the raw material (e.g., Cement, Steel)
quantityDoubleCurrent site-specific stock level
minThresholdDoubleLevel at which low-stock alerts trigger
siteLocationSiteThe specific project site holding this inventory

Infrastructure Health

The maintenance module tracks service requests for assets. Requests move through a status workflow managed by administrators to ensure 100% operational readiness.
